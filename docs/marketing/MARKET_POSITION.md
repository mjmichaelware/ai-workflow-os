# Market Position

AI Workflow OS is for people who want to build real software from the device in their pocket.

Most developer platforms assume a desktop, a large laptop, or a full web console. AI Workflow OS starts from the opposite assumption: the phone should be able to operate the build system.

## Wedge

Phone-first local app factory for Android plus Termux.

## Expansion

- Desktop local runner
- GitHub Codespaces mode
- Cloud runner mode
- SaaS control plane
- Team approvals
- App marketplace
- Generated app template ecosystem
