# UI Packet Architecture Standard

AI Workflow OS console code must be split into decoupled files.

## Required

- HTML shell in web/index.html
- CSS in web/assets/console.css
- JavaScript in web/assets/console.js
- Metadata in web/assets/console.meta.json
- No external fonts or scripts
- Local origin requests only
- Button endpoints must be proven
- Context packets must be reproducible
