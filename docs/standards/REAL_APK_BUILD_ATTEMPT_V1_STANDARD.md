# Real APK Build Attempt v1 Standard

## Rules

- Install build tools outside the repository.
- Never commit SDKs, Gradle caches, APKs, keystores, or passwords.
- Generate an inspectable Gradle Android project under `exports/`.
- Attempt a debug APK build when the local toolchain supports it.
- Record build success or failure honestly.
- Commit only source, reports, docs, and tests.
