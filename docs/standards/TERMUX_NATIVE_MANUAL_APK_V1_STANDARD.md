# Termux-Native Manual APK v1 Standard

## Rules

- Use Termux-native Android tools when SDK Linux binaries are incompatible with Android ARM64.
- Do not commit APKs, keystores, SDKs, Gradle caches, or passwords.
- Generate debug signing material only inside runtime exports.
- Copy built APK to device Downloads or tmp.
- Report build status honestly.
- Keep the app pointed at the local operator URL.
