
# Mobile Command Surface v7 Standard

The phone UI must not hide core navigation.

## Requirements

- Full mobile command menu with every tab.
- Bottom nav may be partial, but the command menu must expose the whole system.
- Mobile layout must not require the left rail.
- Motion must be obvious but respect reduced-motion settings.
- Browser mode and installed app mode may differ, but both must stay usable.
- No arbitrary shell execution from the browser.
