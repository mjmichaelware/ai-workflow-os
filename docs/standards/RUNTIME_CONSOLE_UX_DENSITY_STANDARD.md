
# Runtime Console UX Density Standard

The console must provide a dense, data-rich, local-first operator view.

## Requirements

- Read-only inline runtime terminal.
- Verbose proof surface.
- Long-form About, Privacy, and License pages from decoupled data.
- Dense cards for proof executors, runtime assets, generated apps, and safety state.
- No arbitrary browser-submitted shell execution.
- No secrets printed.
- No broad permission mode.
