
# Editorial UX v6 Standard

Long-form pages must be meaningful, non-repetitive, and decoupled from code.

## Rules

- About, Privacy, and License content must live in local structured data.
- Sections must use unique titles and unique bodies.
- The UI may be dense, but it must not repeat filler text.
- Native app mode and browser tab mode should be visually distinguishable.
- The browser must not execute arbitrary shell commands.
- Visual polish must remain local-only and test-backed.
