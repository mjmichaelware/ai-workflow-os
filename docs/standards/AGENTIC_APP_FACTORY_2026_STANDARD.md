# Agentic App Factory 2026 Standard

Software development is moving toward repository-level agent configuration, agent-authored pull requests, multi-agent coding workflows, and intent-first development.

AI Workflow OS aims to get ahead of that direction by combining:

- local-first phone execution through Termux
- repo-level agent instructions
- approved prompt bridge
- CLI capability inventory
- generated app manifests
- tests before publish
- phone PWA export
- clean public GitHub collaboration
- future desktop and SaaS portability

## Boundary

The browser sends intent.
Termux performs approved material work.
GitHub stores clean public source.
Secrets remain outside source.
