# Brand Neutral Console Standard

AI Workflow OS must look professional without imitating another company product or using another company product name as its interface style.

## Rules

- Do not describe the UI as another company product style.
- Do not use phrases such as Google Cloud style, Google Cloud Console, or Google-style.
- Use neutral terms such as infrastructure-grade, device-native console, local operator, control surface, and public operator.
- Integration names are allowed only when describing real integrations, tools, or capability inventory.
- Public text must stay clean, professional, and contributor-safe.
