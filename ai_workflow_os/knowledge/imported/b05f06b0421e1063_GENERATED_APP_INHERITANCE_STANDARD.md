# Generated App Inheritance Standard

Every generated app must inherit these rules from AI Workflow OS:

- health endpoint or health command
- local proof log
- visible button result and error output
- no fake proof rule
- no secret print rule
- no module package name collision rule
- no fragile paste rule
- no heredoc core-build rule
- runtime proof requirement
- import proof requirement
- generated manifest

A generated app is not valid until compile, runtime health, button or API path, and proof output all pass.
